# THANTHI-ONE-MOVIE-WEBSITE

The Thanthi One Movie Website is a responsive web application inspired by the layout and structure of Thanthi One movie/entertainment section.
This project is developed for educational purposes to demonstrate frontend development skills, structured media content presentation, and modern UI design for a movie-based website.
The website showcases movie details, trailers, cast information, and entertainment news in an organized and user-friendly interface.
🚀 Features
🎥 Featured Movie Banner Section
🎞️ Latest Movie Listings
👥 Cast & Crew Details
📺 Embedded Trailer Section
📰 Entertainment News Section
📱 Fully Responsive Design (Mobile, Tablet & Desktop)
🌐 Smooth Navigation & Clean Layout
🛠️ Technologies Used
HTML5
CSS3
JavaScript (if applicable)
Flexbox / CSS Grid
Responsive Web Design
🎯 Project Objective
The objective of this project is to design a modern movie/entertainment website layout and strengthen skills in:
Media content structuring
Responsive UI development
Interactive elements
Clean and professional layout design
